# Audit Log Pro

**Compliance-ready audit logging.**

Structured, searchable, and tamper-evident audit logs for every economic event.

## Features

- **Structured Events** — JSON format
- **Searchable** — Query by type/date
- **Tamper-Evident** — Hash verification

## Installation

```bash
uv pip install git+https://github.com/vindicta-platform/Audit-Log-Pro.git
```
